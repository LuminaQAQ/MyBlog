1. 变量定义 ​

sass
// 1. 变量定义
$color: #ccc

// 2. 将局部变量设置为全局变量
p {
    $abc: center !global;
}

div {
    text-align: $abc;
}

2. 父选择器(&) ​

sass
// 1. 伪类样式
div {
   &:hover {
       background-color: $color;
   }
}

// 2. 前面部分与后代样式同名
.main-dialog {
  &-text {
      background-color: $color;
  }
}

3. 属性嵌套 ​

4. 继承: @extend ​

sass
// 1.使用 .或# 本身的类, 会被编译在最终的css中
@extend .class;
@extend #id;

// 2. 占位符: %; 定义的变量不会被编译至最终产物中, 只会是用时才会出现在子类
@extend %class

(10px / 2px) // res: 5 (单位消失)

(10px / 2) // res: 5px

18. @for循环 ​

```sass
$ball-nums: 3;

@for $i from 0 through $ball-nums {
  .ball#{$i} {
    $animation-time: 1s;
    $animation-delay: $animation-time / 4;

    left: $i * 2rem;
    animation: jump infinite 2s;
    animation-delay: $i * $animation-delay;
  }
}

@keyframes jump {
  50% {
    top: 0;
  }
}
```

19. @each循环 ​

```sass
$alert-status: normal success warning;

@each $key, $val in $alert-status {
  .alert-#{$k} {
    background-image: url(../images/#{$key}.png);
  }
}
```

20. @while循环 ​

```sass
$end: 0;

@while $end < 7 {
  .ball#{$end} {
    top: $end * 2;
  }

  $end: $end + 1;
}
```
